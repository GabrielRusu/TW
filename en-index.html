    INTRODUCTION

    Project NEDa

    This project uses the available data regarding the Nepalese earthquakes from 2015.

    Using this data we'll be able to more efficiently distribute the available resources to the people in the areas that were affected the most.

    The administrator is able to update this data in real time so that a clear and well documented picture is used in order to make sure that the help will be efficiently distributed.





























    DETAILS

    Datele sunt luate de pe platforme oficiale si transpuse intr-un asa mod incat sa fie usor accesibile oricarei persoane care doreste vizualizarea acestor date

    Exista mai multe moduri de vizualizare, printre care numaram harta,chart-ul , tabelul si graficul.

    Fiecare mod de vizualizare are optiunea de exporta datele afisate la momentul apasarii butonului de exportare.

    Datele se pot exporta in diferite extensii (.PNG, .JPG, .JSON etc)
